# Surrogate mouillé/sec — endpoint d'étendue (k combiné)

POD hauteur combiné sur tout le vocabulaire (un seul jeu de snapshots). k mesuré à erreur L2 globale ≤ 2%, même métrique par-scénario et combiné (apples-to-apples).

- scénarios : **14** (pesés run-up, positions étalées)
- k par-scénario : médian **5**, somme **84** ; N² = 4096
- **k combiné @2% = 46** (55% de la somme, 1.1% de N²)
- courbe de saturation nested : n=1 → k=6 | n=2 → k=6 | n=4 → k=11 | n=8 → k=19 | n=14 → k=46
- incrément marginal (dernier palier) : ~4.5 modes/scénario

## Verdict d'étendue

BORNÉE (pour vocabulaire borné) : k combiné=46 ≪ N²=4096 (base linéaire tractable, 1.1% des DDL), 55% de la somme 84 (sous-additif -> partage réel). Coût marginal ~4.5 modes/scénario (modéré ; il REMONTE pour les régimes nets dam-break/bore, plus divers que les run-up -> PAS un plateau plat). Un vocabulaire BORNÉ reste donc tractable (~few modes/régime) ; l'encodeur de base N'est PAS forcé sur la représentation. Il ne le serait qu'avec une étendue NON bornée (centaines de régimes distincts), absente du livrable.
